# Tennis Analysis

This folder contains analysis scripts and visualizations for Tennis predictions.

## Structure

```
analysis/
├── eda.py              # Exploratory Data Analysis
├── feature_analysis.py # Feature importance and correlation
├── model_evaluation.py # Model performance analysis
└── graphs/             # Generated visualizations
```

## Datasets Used

- **guillemservera/tennis-atp-world-tour** - ATP tour match data
- **ehallmar/tennis-data** - Historical tennis match data  
- **valentinfamworthy/tennis-atp-tour** - ATP rankings and results
