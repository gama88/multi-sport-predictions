# NBA Analysis

This folder contains analysis scripts and visualizations for NBA predictions.

## Structure

```
analysis/
├── eda.py              # Exploratory Data Analysis
├── feature_analysis.py # Feature importance and correlation
├── model_evaluation.py # Model performance analysis
└── graphs/             # Generated visualizations
```

## Datasets Used

- **nathanlauga/nba-games** - NBA Games Data (2003-2022)
- **wyattowalsh/basketball** - Extensive NBA data
- **justinas/nba-players-data** - Player statistics 1996-2022
