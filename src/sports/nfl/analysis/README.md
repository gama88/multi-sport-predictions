# NFL Analysis

This folder contains analysis scripts and visualizations for NFL predictions.

## Structure

```
analysis/
├── eda.py              # Exploratory Data Analysis
├── feature_analysis.py # Feature importance and correlation
├── model_evaluation.py # Model performance analysis
└── graphs/             # Generated visualizations
```

## Datasets Used

- **maxhorowitz/nflplaybyplay2009to2016** - NFL Play by Play Data
- **tobycrabtree/nfl-scores-and-betting-data** - NFL Betting Data
- **cviaxmiern/nfl-team-stats-20022019-espn** - Team Statistics
