# Soccer/Football Analysis

This folder contains analysis scripts and visualizations for Soccer predictions.

## Structure

```
analysis/
├── eda.py              # Exploratory Data Analysis
├── feature_analysis.py # Feature importance and correlation
├── model_evaluation.py # Model performance analysis
└── graphs/             # Generated visualizations
```

## Datasets Used

- **davidcariboo/player-scores** - Football/Soccer match data
- **hugomathien/soccer** - European Soccer Database
- **technika148/football-statistics** - Comprehensive football stats
- **secareanualin/football-events** - Match events data
