# NCAA Basketball Analysis

This folder contains analysis scripts and visualizations for NCAA Basketball predictions.

## Structure

```
analysis/
├── eda.py              # Exploratory Data Analysis
├── feature_analysis.py # Feature importance and correlation
├── model_evaluation.py # Model performance analysis
└── graphs/             # Generated visualizations
```

## Datasets Used

- **andrewsundberg/college-basketball-dataset** - College Basketball Data
- **nishaanamin/march-madness-data** - March Madness Tournament Data
- **ncaa/ncaa-basketball** - Official NCAA Data
