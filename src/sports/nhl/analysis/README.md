# NHL Analysis

This folder contains analysis scripts and visualizations for NHL predictions.

## Structure

```
analysis/
├── eda.py              # Exploratory Data Analysis
├── feature_analysis.py # Feature importance and correlation
├── model_evaluation.py # Model performance analysis
└── graphs/             # Generated visualizations
```

## Datasets Used

- **martinellis/nhl-game-data** - NHL Games Data
- **open-source-sports/professional-hockey-database** - Hockey Database
- **zingbretsen/nhl-play-by-play** - Play by Play Data
