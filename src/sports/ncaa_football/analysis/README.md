# NCAA Football Analysis

This folder contains analysis scripts and visualizations for NCAA Football predictions.

## Structure

```
analysis/
├── eda.py              # Exploratory Data Analysis
├── feature_analysis.py # Feature importance and correlation
├── model_evaluation.py # Model performance analysis
└── graphs/             # Generated visualizations
```

## Datasets Used

- **tylerr1990/ncaafb-team-game-stats-2014-2022** - CFB Team Game Stats
- **mhixon/college-football-statistics** - College Football Statistics
