# MLB Analysis

This folder contains analysis scripts and visualizations for MLB predictions.

## Structure

```
analysis/
├── eda.py              # Exploratory Data Analysis
├── feature_analysis.py # Feature importance and correlation
├── model_evaluation.py # Model performance analysis
└── graphs/             # Generated visualizations
```

## Datasets Used

- **open-source-sports/baseball-databank** - Baseball Databank
- **pschale/mlb-pitch-data-20152018** - Pitch Level Data
- **seanlahman/the-history-of-baseball** - Historical Data
